## 🇷🇺 Русский

### О проекте

Веб-интерфейс CRM-системы для салона красоты **«Элит Эстетика»**. Реализован на чистом HTML5, CSS3 и JavaScript без использования серверной части. Позволяет клиентам записываться к мастерам онлайн, просматривать каталог услуг и мастеров.

### 📄 Страницы

| Страница | Файл | Описание |
| --- | --- | --- |
| Главная | `index.html` | Баннер, услуги, мастера, акции, контакты |
| Вход и регистрация | `auth.html` | Форма входа и регистрации с валидацией |
| Запись к мастеру | `appointment.html` | Пошаговая форма записи с выбором мастера, услуги и времени |

### ✨ Функционал

- ✅ Адаптивный дизайн (Bootstrap 5)
- ✅ Фиксированная навигация
- ✅ Форма входа и регистрации с валидацией
- ✅ Пошаговая запись к мастеру (3 шага)
- ✅ Выбор мастера, категории услуги, даты и времени
- ✅ Живое обновление итоговой суммы
- ✅ Занятые слоты времени
- ✅ Модальное окно подтверждения записи

## 🇬🇧 English

### About

A front-end CRM interface for beauty salon **«Элит Эстетика»** (Elite Esthetics). Built with pure HTML5, CSS3 and vanilla JavaScript — no backend, no build tools required. Clients can browse services and masters, register, and book appointments online.

### 📄 Pages

| Page | File | Description |
| --- | --- | --- |
| Home | `index.html` | Hero banner, services, masters, promotions, contacts |
| Login & Register | `auth.html` | Login and registration forms with validation |
| Book Appointment | `appointment.html` | 3-step booking flow: master → service → time slot |

### ✨ Features

- ✅ Fully responsive (Bootstrap 5)
- ✅ Fixed navigation bar
- ✅ Login & registration with inline validation
- ✅ 3-step appointment booking
- ✅ Master selector, service categories, date & time picker
- ✅ Live price summary sidebar
- ✅ Pre-disabled booked time slots
- ✅ Confirmation modal on booking submit

### 🛠️ Tech Stack

| Technology | Purpose |
| --- | --- |
| HTML5 | Page structure |
| CSS3 + CSS Variables | Styling and theming |
| Bootstrap 5 | Responsive grid and UI components |
| Bootstrap Icons | Icon set |
| Vanilla JavaScript | Interactivity and form validation |
| Google Fonts | Playfair Display (headings) + Inter (body) |

### 🚀 Getting Started

```bash
# No installation needed — just open in browser
open index.html
```
